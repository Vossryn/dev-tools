# Dev Tools

A growing collection of privacy-friendly, browser-first developer tools. Build, test, and ship faster with utilities that run entirely on your device. No accounts, no uploads—just helpful tools you control.

## Features

This application includes the following tools:

- **JSON Parser & Linter**: Validate, format, and inspect JSON by pasting or uploading a file.
- **YAML Linter & Formatter**: Validate, prettify, and inspect YAML documents—including multi-document files—in the browser.
- **CSS Parser, Linter & Formatter**: Validate, format, and minify modern CSS with native nesting support.
- **Image Converter**: Convert images between PNG, JPEG, and WebP directly in your browser.
- **Pixel Converter**: Translate measurements between px, rem, and em with a custom root size.
- **CSS Color Converter**: View synchronized color values across hex, RGB, HSL, Lab, OKLab, and more.
- **SVG Optimizer**: Upload and clean SVGs with SVGO presets, preview, and export.
- **Contrast Checker**: Calculate contrast ratios and check WCAG compliance for accessibility.
- **Regex Tester**: Real-time regex tester with explainers and shareable test cases for frontend debugging.
- **Markdown Preview**: Side-by-side Markdown editor with HTML export for docs/snippets.

## Tech Stack

This project is built using the following technologies:

- **[React](https://react.dev/)**: The library for web and native user interfaces.
- **[TanStack Start](https://tanstack.com/start)**: Full-stack React framework.
- **[TanStack Router](https://tanstack.com/router)**: Type-safe routing for React applications.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework.
- **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling.
- **[Vitest](https://vitest.dev/)**: Blazing Fast Unit Test Framework.

## Getting Started

To run this application:

```bash
npm install
npm run dev
```

## Building For Production

To build this application for production:

```bash
npm run build
```

## Testing

This project uses [Vitest](https://vitest.dev/) for testing. You can run the tests with:

```bash
npm run test
```
